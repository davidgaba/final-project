Oh, dear!

The page you are looking for cannot be found!

Include a prominent explanation of the page not found status.

Maybe provide an amusing graphic to reduce the unpleasant nature of a page-not-found user experience.

Be sure a 404 page includes a full site navigation bar (with links to pages that will be found)!

Locate this 404.html file in your server root folder.

Use server root paths for this page's relative paths (paths for CSS, images, etc).

Use .htaccess to implement error handling for Page Not Found.

#to affect the entire site, place .htaccess file 
#in the server root directory
#add error handling in the .htaccess file, eg:

ErrorDocument 404 /root-based-path/to/404.html

Place the .htaccess file in your server root folder.

Trigger a 404 error by making a request over http:// using your locahost server for a file or folder that does not exist in your server root.

This page should never be explicitly requested by name. This page should only be a server response when an actual error 404 'page not found' occurs, and the .htaccess has redirected the user agent to this page.
